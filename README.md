## ✅ Task 1 — Responsive Landing Page

**Objective:** Build a visually stunning, fully responsive landing page representing a startup/product launch.

### What's Built
- **Header & Navigation** — Fixed glassmorphism navbar, dropdown under "Services", mobile hamburger menu
- **Hero Section** — Animated gradient background, *"Build Smarter with MainCrafts"* heading, CTA buttons, stats row
- **Features Section** — 4 cards (Fast, Responsive, Scalable, Secure) with FontAwesome icons and hover effects
- **Services Section** — Two-column layout with animated orb and service list
- **Footer** — 4-column grid, social links, Copyright © 2025, Privacy Policy | Terms | Contact

### Tech Used
| Category | Details |
|----------|---------|
| **Fonts** | Sora + Inter via Google Fonts |
| **Icons** | Font Awesome 6.5 |
| **Colors** | Violet `#7C3AED` + Cyan `#06B6D4` on dark `#0A0A0F` |
| **Effects** | Glassmorphism, gradients, scroll-reveal, hover animations |
| **Responsive** | Mobile → Tablet → Desktop |
